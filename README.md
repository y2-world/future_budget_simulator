## 概要

家計シミュレーターは、月ごとの支出・収入の想定額を入力することで、支払いごとの口座残高推移を確認できるWebアプリケーションです。将来の貯金・預金残高の見通しを立て、家計管理をサポートします。

### 主な特徴

- 月ごとの収支計画を編集可能
- 支払いイベント単位での残高推移を表示
- グラフによる視覚的な残高推移の確認
- CSV読み込み・書き出し機能（将来実装予定）
- Zaim API連携（将来実装予定）

## 主な機能

### 1. 初期設定
- 初期口座残高の入力
- シミュレーション開始日の設定
- シミュレーション期間の設定（12ヶ月、24ヶ月など）

### 2. 月次計画入力
- 給与
- 食費
- 家賃
- レイク返済
- クレカ引落額
- 定期預金額
- ボーナス（該当月のみ）
- マネーアシスト返済（該当月のみ）
- その他（光熱費、交通費、娯楽費など）

### 3. シミュレーション実行
- 月ごとのイベントを日付順に整理
- 支払いごとに残高を計算
- 計算結果をテーブルとグラフで表示

### 4. 結果表示
- 支払いイベント一覧（日付、イベント名、金額、残高）
- 月ごとの残高推移グラフ（折れ線グラフ）
- 収支サマリー

### 5. データ管理
- 月次計画の保存・更新
- CSV エクスポート（将来実装予定）
- CSV インポート（将来実装予定）

## セットアップ

### 前提条件
- Python 3.8以上がインストールされていること

### インストール手順

1. リポジトリのクローン
```bash
cd future_budget_simulator
```

2. 仮想環境の作成と有効化
```bash
python -m venv venv
source venv/bin/activate  # Windowsの場合: venv\Scripts\activate
```

3. 依存パッケージのインストール
```bash
pip install django
```

4. データベースのマイグレーション
```bash
python manage.py makemigrations
python manage.py migrate
```

5. 管理者ユーザーの作成
```bash
python manage.py createsuperuser
```

6. 開発サーバーの起動
```bash
python manage.py runserver
```

7. ブラウザでアクセス
```
http://127.0.0.1:8000/
```

## 使用方法

### 基本的な利用フロー

1. **初期設定**
   - 管理画面またはWebUIから初期口座残高を入力
   - シミュレーション開始日と期間を設定

2. **月次計画入力**
   - 月を選択
   - 各項目（給与、食費、家賃など）の想定額を入力
   - 保存ボタンをクリック

3. **シミュレーション実行**
   - 「計算」ボタンをクリック
   - システムが自動的に残高推移を計算

4. **結果確認**
   - テーブルで支払いイベントごとの残高を確認
   - グラフで視覚的に残高推移を確認
   - 必要に応じて月次計画を修正して再計算

5. **データ保存**
   - 結果をCSVでエクスポート（将来実装予定）
